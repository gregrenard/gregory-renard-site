# gregory-renard

Site personnel de Gregory Renard — architecte IA.

## Visualiser le site

GitHub n'affiche pas le rendu HTML directement. Pour voir le site en ligne, activez **GitHub Pages** :

1. Repo \`Settings\` → \`Pages\`
2. Source : branche \`main\`, dossier \`/ (root)\`
3. Le site sera disponible sur \`https://gregrenard.github.io/gregory-renard/\`

La page d'accueil (\`index.html\`) redirige automatiquement vers \`Gregory Renard - Home.dc.html\`.
